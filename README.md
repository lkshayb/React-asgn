# React Spreadsheet UI Assignment
This project is a React-based spreadsheet-like table UI built using:
- React 18
- TypeScript (strict mode)
- Tailwind CSS
- @tanstack/react-table v8

### Setup Instruction
1. Clone Repository
```
git clone https://github.com/lkshayb/React-asgn.git
cd React-asgn
```
2. Install Dependencies
```
npm install

```
3. Run the Development Server
```
npm run dev

```
4. Run Lint & Type-check test
```
npm run lint
npm run type-check
```
